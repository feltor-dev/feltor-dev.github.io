Detailed Description

template<class Index, class Vector>
struct dg::NearestNeighborComm< Index, Vector >

Communicator for asynchronous nearest neighbor communication.

exchanges a halo of given depth among neighboring processes in a given direction (the corresponding gather map is of general type and the communication can also be modeled in GeneralComm, but not BijectiveComm or SurjectiveComm )

Template Parameters
Indexthe type of index container (must be either thrust::host_vector<int> or thrust::device_vector<int>)
Vectorthe vector container type must have a resize() function and work in the thrust library functions ( i.e. must a thrust::host_vector or thrust::device_vector)

This test can be used to avoid the gather operation alltogether in e.g. the construction of a MPI distributed matrix.

◆ size()

template<class Index, class Vector>
unsigned dg::NearestNeighborComm< Index, Vector >::size ( ) const
inline

The size of the local buffer vector w = local map size.

Consider that both the source vector v and the buffer w are distributed across processes. In Feltor the vector v is distributed equally among processes and the local size of v is the same for all processes. However the buffer size might be different for each process.

Returns
buffer size
Note
may return 0 to indicate that no MPI communication is needed
we assume that, contrary to size(), the vector size is always the local size of a dg::MPI_Vector
