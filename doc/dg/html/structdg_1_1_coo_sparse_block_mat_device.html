Detailed Description

template<class value_type>
struct dg::CooSparseBlockMatDevice< value_type >

Coo Sparse Block Matrix format device version.

This class holds a copy of a CooSparseBlockMat on the device, which may be gpu or omp depending on the THRUST_DEVICE_SYSTEM macro. It does the same thing as the host version with the difference that it applies to device vectors.
