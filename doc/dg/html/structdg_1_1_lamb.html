Detailed Description

Functor returning a Lamb dipole

\[ f(x,y) = \begin{cases} 2\lambda U J_1(\lambda r) / J_0(\gamma)\cos(\theta) \text{ for } r<R \\ 0 \text{ else} \end{cases} \]

.

with \( r = \sqrt{(x-x_0)^2 + (y-y_0)^2}\), \( \theta = \arctan_2( (y-y_), (x-x_0))\), \(J_0, J_1\) are Bessel functions of the first kind of order 0 and 1 and \(\lambda = \gamma/R\) with \( \gamma = 3.83170597020751231561\)
