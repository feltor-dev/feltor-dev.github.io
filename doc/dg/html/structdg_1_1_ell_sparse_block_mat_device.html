Detailed Description

template<class value_type>
struct dg::EllSparseBlockMatDevice< value_type >

Ell Sparse Block Matrix format device version.

This class holds a copy of a EllSparseBlockMat on the device, which may be gpu or omp depending on the THRUST_DEVICE_SYSTEM macro. It can be applied to device vectors and does the same thing as the host version
