Detailed Description

template<class Matrix, class container>
struct dg::MultiMatrix< Matrix, container >

Struct that applies given matrices one after the other.

\[ y = M_{N-1}(...M_1(M_0x))\]

where \( M_i\) is the i-th matrix

Template Parameters
MatrixAny class for which a specialization of MatrixTraits exists and which fullfills the requirements of the there defined Matrix policy derived from AnyMatrixTag or SelfMadeMatrixTag. The Matrix type can for example be one of:
  • container: A container acts as a diagonal matrix.
  • dg::HMatrix and dg::IHMatrix with dg::HVec or std::vector<dg::HVec>
  • dg::DMatrix and dg::IDMatrix with dg::DVec or std::vector<dg::DVec>
  • dg::MHMatrix with dg::MHVec or std::vector<dg::MHVec>
  • dg::MDMatrix with dg::MDVec or std::vector<dg::MDVec>
  • In case of SelfMadeMatrixTag only those blas2 functions that have a corresponding member function in the Matrix class (e.g. symv( const container&, container&); ) can be called.
If a container has the VectorVectorTag, then the Matrix is applied to each of the elements.
See also
List of Matrix Traits specializations
Template Parameters
containerAny class for which a specialization of VectorTraits exists and which fulfills the requirements of the there defined data and execution policies derived from AnyVectorTag or AnyPolicyTag. For example, this is one of
  • dg::HVec, dg::DVec, dg::MHVec or dg::MDVec
  • std::vector<dg::DVec>, std::array<dg::MHVec, 3>, std::array<double, 4> ...
