Detailed Description

These routines form the heart of our geometry free numerical algorithms. They are called by our geometric operators like the Poisson bracket.

Parameters
tinput Tensor
in0(input) first component (restricted)

out0(output) first component (restricted)

Attention
aliasing only allowed between out1 and in1
Note
Currently required memops:
  • 10(9) reads + 2 writes if all values in t are set; (-1 read if alias is used)
  • 6(5) reads + 2 writes if t is diagonal; (-1 read if alias is used)
  • 4 reads + 2 writes if t is empty and no alias
  • 2 reads + 1 writes (= 1 copy) if t is empty and out1 aliases in1
