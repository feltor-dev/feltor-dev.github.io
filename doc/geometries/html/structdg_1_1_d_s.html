Detailed Description

template<class ProductGeometry, class IMatrix, class Matrix, class container>
struct dg::DS< ProductGeometry, IMatrix, Matrix, container >

Class for the evaluation of a parallel derivative.

This class discretizes the operators \( \nabla_\parallel = \mathbf{b}\cdot \nabla = b_R\partial_R + b_Z\partial_Z + b_\phi\partial_\phi \), \(\nabla_\parallel^\dagger\) and \(\Delta_\parallel=\nabla_\parallel^\dagger\cdot\nabla_\parallel\) in arbitrary coordinates

Template Parameters
ProductGeometrymust be either aProductGeometry3d or aProductMPIGeometry3d or any derivative
IMatrixThe type of the interpolation matrix -dgIHMatrix, or dg::IDMatrix, dg::MIHMatrix, or dg::MIDMatrix
MatrixThe matrix class of the jump matrix -dgHMatrix, or dg::DMatrix, dg::MHMatrix, or dg::MDMatrix
containerThe container-class on which the interpolation matrix operates on -dgHVec, or dg::DVec, dg::MHVec, or dg::MDVec
